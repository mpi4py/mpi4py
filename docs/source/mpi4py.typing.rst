mpi4py.typing
=============

.. module:: mpi4py.typing
   :synopsis: Typing support.

.. versionadded:: 4.0.0

This module provides :term:`type aliases <type alias>` used to add
:term:`type hints <type hint>` to the various functions and methods
within the :mod:`~mpi4py.MPI` module.
